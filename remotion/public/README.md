# Static assets

Place the Cellulift logo here, unchanged, named exactly:

    cellulift-logo.png

A transparent-background PNG (or SVG) works best so it sits cleanly on the
ivoire background. The compositions reference it via `staticFile("cellulift-logo.png")`
and animate *around* it (fade, gentle scale, pulsation, particles) — the logo
image itself is never altered.
